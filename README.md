# Booth Shop Blocker

BOOTHで特定のショップをブロックして、快適にブラウジングできるFirefox拡張機能です。

## 機能

- ✅ **ショップブロック**: ショップページから簡単にブロック可能
- 📝 **ブロック理由メモ**: なぜブロックしたのか記録できます
- 🔍 **自動非表示**: ブロックしたショップの商品が検索結果や一覧から自動的に非表示
- 📋 **管理画面**: ブロック中のショップ一覧と解除が可能
- ⚡ **リアルタイム反映**: ブロック/解除がすぐに反映されます

## 使い方

### 1. ショップをブロック

1. BOOTHのショップページ（例: `https://shopname.booth.pm/`）にアクセス
2. ページ上部に表示される「🚫 このショップをブロック」ボタンをクリック
3. ブロック理由を入力（任意）
4. ブロック完了！

### 2. ブロック中のショップを確認

1. ブラウザのツールバーにある拡張機能アイコンをクリック
2. ブロック中のショップ一覧が表示されます
3. ショップ名、ブロック理由、ブロック日時を確認できます

### 3. ブロックを解除

以下のいずれかの方法で解除できます：

**方法1: ポップアップから**
1. 拡張機能アイコンをクリック
2. 解除したいショップの「🔓」ボタンをクリック

**方法2: ショップページから**
1. ブロック中のショップページにアクセス
2. 「🔓 ブロック解除」ボタンをクリック

## インストール方法

### 開発版（手動インストール）

1. このリポジトリをクローンまたはダウンロード
2. Firefoxで `about:debugging` を開く
3. 「このFirefox」をクリック
4. 「一時的なアドオンを読み込む」をクリック
5. `manifest.json` を選択

### リリース版（準備中）

Firefox Add-onsストアで公開予定です。

## 技術仕様

- **対応サイト**: `*.booth.pm`
- **ストレージ**: `browser.storage.local`（ブロック情報を永続化）
- **動的監視**: `MutationObserver`（スクロール読み込みにも対応）
- **Manifest**: Version 2（Firefox安定版対応）

## データ構造

ブロック情報は以下の形式でローカルストレージに保存されます：

```json
{
  "blockedShops": {
    "shopname": {
      "name": "ショップ名",
      "reason": "ブロック理由",
      "timestamp": 1700000000000
    }
  }
}
```

## プライバシー

- ブロック情報はローカル（お使いのブラウザ内）にのみ保存されます
- 外部サーバーへのデータ送信は一切行いません
- BOOTH以外のサイトにはアクセスしません
